About Our Contact Management System

Our Contact Management System (CMS) is a web-based application built to help users efficiently manage their personal or professional contacts. It provides a secure, organized, and user-friendly platform to store, view, edit, and categorize contact details.

📌 Key Features

• Create, edit, and delete contacts

• Group-based contact organization

• Profile image uploads using Cloudinary

• Dark/Light theme toggle

• Secure login with Spring Security & OAuth

• Responsive and elegant UI using Tailwind CSS

🛠 Tech Stack

• Spring Boot, Java, Hibernate

• MySQL database

• Tailwind CSS, HTML5, Thymeleaf

• AJAX, Google Cloud Console, Cloudinary

• Spring Security, OAuth2 authentication
